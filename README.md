# JSP를 이용한 회원들끼리 맛집을 공유하는 게시판
## 1. 개요
게시판의 주된 목적은

회원들끼리 음식점을 올려서 평가할 수 있게합니다.

평가는 좋아요, 그럭저럭, 맛없어요 버튼으로 가능합니다.

댓글로 맛있는 메뉴나 의견을 나눌 수 있습니다.
